Utilizando itálico, neste termo com a tag <i>(Não Semântica)

Utilizando ênfase, neste termo com a tag <em>(Semântica)

Utilizando deletado, neste termo com a tag <del>(Semântica)

Sublinhado / Inserido

Utilizando sublinhado, neste termo com a tag <u>(Não Semântica)

Utilizando inserido, neste termo com a tag <ins>(Semântica)

Utilizando subscrito, para coisas do tipo H2, O com a tag <sub>
